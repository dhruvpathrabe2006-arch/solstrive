# SolStrive

Monorepo: Next.js frontend + Express/TypeScript backend.

## Apps
- frontend: Next.js 14 + Tailwind (App Router)
- backend: Express + TypeScript + Mongoose + JWT

## Getting Started

### Frontend

1. cd frontend
2. npm run dev

### Backend

1. cd backend
2. npm run dev

Set env in `backend/.env` (MONGO_URI, JWT_SECRET).
